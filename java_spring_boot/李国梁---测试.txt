	Mybatis传参中，#{*}、${*}的区别？

               答：#{*}是预编译处理，${*}是字符替换。 在使用 #{*}时，MyBatis 会将 SQL 中的 #{*}替换成“?”，
                       配合 PreparedStatement 的 set 方法赋值，这样可以有效的防止 SQL 注入，保证程序的运行安全。
	
              简述Restful规则？

             答：1.C-S框架
                    2.无状态
                    3.统一接口
                    4.一致的数据结构
                    5.系统分层
                    6.可缓存

	简述Git使用流程，写几个常用Git命令？

                     常用命令：git push origin master    
                                      git  clone 远程地址

	简述shiro两大功能实现流程；

               答:认证执行流程：构造SecuriManager--->Subject.login()认证
                                          --->SecuriManager.login()执行认证---->authenticator执行认证
                                         --->realm获取验证信息
                     授权流程：构造SecuriManager--->subject.isPermitted()授权
                                     ----->SecuriManager.isPermitted()授权--->authorizer授权
                                         --->realm获取资源权限
